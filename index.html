Generic Monte-Carlo simulations

The generic template library mocasinns is a user-friendly framework for using various Monte-Carlo algorithm on user defined systems. It integrates the simulations, storage and analysis of the data. Included in the mocasinns package is the include library gespinst for simulating arbitrary configurations of spin systems.

What works?

  • Metropolis algorithm
  • Entropic sampling
  • Wang-Landau simulations
  • Optimal ensemble sampling
  • Simple analysis tools (e.g. Jackknife and Bootstrap)
  • Basic templates for usage in user code (observables with several components etc.)

What's to come?

  • Parallel Tempering
  • Simulated Annealing
  • Rejection-free / continous-time algorithms
  • Framework to organize and store simulation parameters and results
  • A more sophisticated analysis framework
  • Python interface The contributors are happy for hints about other algorithms or modifications of existing algorithms that can be implemented in mocasinns, as well as other analysis methods.

Which projects use mocasinns?

The following projects are using mocasinns at present:

  • gespinst: GEneralized SPIn Network Toolkit, generic library for simulating arbitrary spin configurations like Ising model on various networks. This library is packaged with mocasinns and is used by mocasinns for testing
  • hasi-geco: HArd Spheres In GEometrical COnfinement
  • mocatie: MOnte-CArlo Take It Easy (a popular german board game) If you want your own project using mocasinns listed here or if you know about another project using mocasinns, please feel free to contact the authors. We then will update the list.
